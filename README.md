## ðŸ’¿ Setup

### Prerequisites

- Node v16 or above
- NPM

### Installation steps

First, clone the repository with the following command:

```sh
git clone git@github.com:vcheeney/TriTrainingLog.git
```

### Usage

Run the frontend:

```sh
cd frontend
npm run dev
```

Run the backend:

```sh
cd backend
npm run start:dev
```
